EN_PB_API_STATE PB_Init ( PB_Init_t *  p_pb_config_struct)

Initializes the state of the pin connected to the push button.

  • The function declares a DIO_Init_t variable and extract the port and the pin from the p_pb_config_struct, with a constant value for the 'direction' member as DIO_INPUT.
  • Then it calls DIO_Init function.
